# Robot-Sumo-TEJAS
Design and development of a 4WD wedge-based sumo robot built for SCI-MANIA 2025 at KLS GIT.
Objectives - Design a mechanically robust and compact robot
             Achieve high traction and pushing capability
             Stay within 12V power and 5 kg weight limit
             Fabricate chassis and wedge in-house
             Ensure safety using BMS and master switch.
Technical Specifications - Drive System	4WD 
                           Motors 12V Jhoson DC Geared Motors (300 RPM) 
                           Wheel Diameter	85 mm (bat grip enhanced) 
                           Battery	3S3P Lithium-Ion (11.1V nominal, 6000 mAh) 
                           Chassis Material	1.5 mm Mild Steel 
                           Max Speed	~1.33 m/s 
                           Total Weight	~2.25 kg
Key Features
High-traction wedge design.
Custom 3S3P Lithium-Ion battery pack with BMS.
Torque optimized for 39.24 N traction force.
Total system power ~210 W.
Structural analysis and load calculations included.
In-house fabricated chassis and wedge.
